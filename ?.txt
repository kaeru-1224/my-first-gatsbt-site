3) 에러 
-Adjacent JSX elements must be wrapped in an enclosing tag. Did you want a JSX fragment
const Homepage = () => {
  return <h1> hello!</h1>
          <p>this is first page</p>;
};
이대로 작성하는 경우 발생하는 에러로, 하나의 엘리먼트만 리턴해줄 수있으므로, 감싸는 작업을 해줘야함
const Homepage = () => {
  return (
    <div>
      <h1> hello!</h1>
      <p>this is first page</p>
    </div>
  );
};
그런데 이렇게 작성을했을때  div가 연속적으로 작성되어 보기 어려울경우에는  fragment를 이용해주면되는데 
const Homepage = () => {
  return (
    <>
      <h1> hello!</h1>
      <p>this is first page</p>
    </>
  );
};
이대로 진행해주면되는것
